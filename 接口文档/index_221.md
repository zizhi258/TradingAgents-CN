## 国债收益率曲线利率（日频）

---

接口：us\_tycr
描述：获取美国每日国债收益率曲线利率
限量：单次最大可获取2000条数据
权限：用户积累120积分可以使用，积分越高频次越高。具体请参阅[积分获取办法](https://tushare.pro/document/1?doc_id=13)

**输入参数**

| 名称 | 类型 | 必选 | 描述 |
| --- | --- | --- | --- |
| date | str | N | 日期 （YYYYMMDD格式，下同） |
| start\_date | str | N | 开始日期 |
| end\_date | str | N | 结束日期 |
| fields | str | N | 指定输出字段（e.g. fields='m1,y1'） |

**输出参数**

| 名称 | 类型 | 默认显示 | 描述 |
| --- | --- | --- | --- |
| date | str | Y | 日期 |
| m1 | float | Y | 1月期 |
| m2 | float | Y | 2月期 |
| m3 | float | Y | 3月期 |
| m4 | float | Y | 4月期（数据从20221019开始） |
| m6 | float | Y | 6月期 |
| y1 | float | Y | 1年期 |
| y2 | float | Y | 2年期 |
| y3 | float | Y | 3年期 |
| y5 | float | Y | 5年期 |
| y7 | float | Y | 7年期 |
| y10 | float | Y | 10年期 |
| y20 | float | Y | 20年期 |
| y30 | float | Y | 30年期 |

**接口调用**

```
pro = ts.pro_api()

df = pro.us_tycr(start_date='20180101', end_date='20200327')

#获取1月期和1年期数据
df = pro.us_tycr(start_date='20180101', end_date='20200327', fields='m1,y1')
```
