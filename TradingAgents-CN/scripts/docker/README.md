# Docker Scripts

## 目录说明

Docker容器管理脚本

## 脚本列表

- `docker-compose-start.bat - 启动Docker Compose`
- `start_docker_services.* - 启动Docker服务`
- `stop_docker_services.* - 停止Docker服务`
- `mongo-init.js - MongoDB初始化脚本`

## 使用方法

```bash
# 进入项目根目录
cd C:\code\TradingAgentsCN

# 运行脚本
python scripts/docker/script_name.py
```

## 注意事项

- 确保在项目根目录下运行脚本
- 检查脚本的依赖要求
- 某些脚本可能需要特殊权限
